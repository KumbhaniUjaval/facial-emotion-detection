# Facial Emotion Detection using CNN

This project detects human facial emotions in real-time using deep learning and computer vision.

## Technologies
- Python
- TensorFlow / Keras
- OpenCV
- MediaPipe
- CNN

## Features
- Real-time emotion detection from webcam
- Image emotion prediction
- Model training using facial datasets

## Project Structure
app.py – Flask web application  
train_emotion_model.py – trains CNN model  
live_emotion_detection.py – webcam emotion detection  
image_prediction.py – predict emotion from image  

## How to Run

1. Install dependencies
pip install -r requirements.txt
2. Run the application
python app.py
